# MeowOS v2.3

A cat-themed operating system that runs entirely inside your browser.

MeowOS started with a simple question and the idea from hackclub:

> What if a cat was put in charge of designing an operating system?

The result is a desktop environment built using HTML, CSS, and JavaScript, complete with windows, apps, wallpapers, notifications, sound effects, and just enough feline influence to make things interesting.

Unlike a typical website, MeowOS behaves more like a real desktop. You boot into a welcome screen, open applications from a dock, manage windows, customize your wallpaper, and explore a collection of built-in apps.

## Features

### Desktop Environment

* Desktop-style interface
* Dock inspired by modern operating systems
* Start menu with search
* Draggable application windows
* Real-time clock
* Notification system

### Personalization

* Multiple built-in wallpapers
* Custom wallpaper uploads
* Persistent settings using local storage

### Built-in Applications

* **Meme Gallery** – because every operating system needs priorities
* **PurrFox** – a browser inside a browser
* **PawPad** – simple note taking
* **MeowTunes** – music player
* **CatPaint** – drawing canvas
* **CatShell** – terminal-inspired interface
* **Fur Files** – file explorer
* **Calc-Cat** – calculator
* **PurrPlanner** – calendar
* **MeowChat** – chat application
* **Catch Yarn** – a small built-in game
* **Settings**
* **About Me**

## Why I Built This

Most web projects try to look like websites.

I wanted to build something that felt like an actual operating system while still running entirely in a browser. The cat theme began as a joke, but eventually became the identity of the project.

The goal was to create something playful, interactive, and slightly ridiculous while experimenting with UI design, window management, local storage, and browser-based desktop experiences.

## Tech Stack

* HTML
* CSS
* JavaScript
* Local Storage API

No frameworks. No libraries. Just browser technology and a questionable amount of cat-related naming.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/Ajmaleee/MeowOS-v2.git
```

Open `index.html` in your browser.

That's it.
